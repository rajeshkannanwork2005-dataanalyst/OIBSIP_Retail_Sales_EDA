# Retail Sales Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to uncover patterns, trends, and useful business insights.

The analysis includes data cleaning, descriptive statistics, time-series analysis, category-wise sales analysis, and data visualization.

---

## Objective
To analyze retail sales data and generate insights that can support business decision-making.

---

## Dataset
Retail Sales Dataset

Features:
- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount

---

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Project Workflow

### 1. Data Loading
- Imported dataset using Pandas

### 2. Data Cleaning
- Checked missing values
- Removed duplicates
- Converted date format

### 3. Exploratory Data Analysis
- Descriptive statistics
- Customer analysis
- Sales trend analysis

### 4. Data Visualization
Generated:
- Sales Trend Line Chart
- Product Category Sales Bar Chart

---

## Key Insights

- Electronics generated the highest total sales.
- Clothing category performed closely behind electronics.
- Sales fluctuated across different dates.
- Dataset required minimal cleaning.

---

## Results
Visualizations were created to understand:
- Total sales amount over time
- Product category contribution
